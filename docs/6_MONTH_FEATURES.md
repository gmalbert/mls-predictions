# MLS Predictor — 6-Month Feature Roadmap

## Month 1: Match Day

- **Today's slate** — All MLS matches today with model win probabilities, O/U prediction, and venue surface indicator.
- **Turf/grass surface badge** — Prominent indicator on every home team's matchup card.
- **Back-to-back flag** — Visual alert when a team played 72 hours ago.
- **Cross-conference badge** — "Cross-conference matchup" indicator on applicable games.

## Month 2: Team Pages

- **Team profile** — ASA xG trend chart, DP roster, recent results, altitude context.
- **Home advantage breakdown** — How much stronger/weaker each team is at home vs. away.
- **Travel distance history** — Season total travel miles for each team.

## Month 3: Betting Tools

- **Value bet finder** — Edge > 3% vs. DraftKings moneyline.
- **Totals analysis** — High-altitude stadiums and turf fields filter for over/under value.
- **Playoff race betting** — Odds to make Western/Eastern Conference playoffs.

## Month 4: Analytics

- **Conference table** — Live Eastern/Western Conference standings with playoff positions.
- **Expansion team tracker** — Historical performance of first-year MLS clubs.
- **Designated player impact** — Season stats with/without each team's DP on the field.

## Month 5: Historical Analysis

- **Model accuracy by feature segment** — Accuracy for turf vs. grass, long-haul vs. local, cross vs. intra-conference.
- **Season recap** — End-of-year accuracy and ROI report.
- **Salary cap tier comparison** — How model accuracy varies across budget tiers.

## Month 6: Automation

- **Matchday email** — Friday morning picks email for the weekend MLS slate.
- **ASA data refresh** — Weekly GitHub Action to pull updated xG from `itscalledsoccer`.
- **Model retraining** — Off-season trigger using full-season data.
